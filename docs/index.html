Free & open source · macOS 14+ · Native Swift & SwiftUI

Everything you loved on Windows.
Now native on your Mac.

No Electron, no cloud, no telemetry. Just a lightweight Swift app and your files.

⚡️

Search as you type

Results appear instantly while you type — file names, paths, indexed volumes, and file lists, ranked Everything-style.

🗄

Local-first index

SQLite in WAL mode with FTS5 candidate seeding handles huge catalogs. Your index never leaves your machine.

🔄

Always up to date

FSEvents watches every indexed root and applies incremental updates the moment files change, move, or disappear.

⌥

Global hotkey palette

Press ⌥ Space anywhere to open a compact floating search palette. Type, hit Enter, done.

🔍

Everything query syntax

Operators, wildcards, regex, fuzzy matching, and hundreds of search functions like ext:, size:, dm:today, dupe:.

💽

External & network volumes

Index mounted local, external, and network volumes, plus EFU-like file lists for offline and NAS catalogs.

🧰

HTTP API & CLI

A local read-only query service and MacThingCLI let scripts and tools search your index in JSON, CSV, TXT, or EFU.

🍎

Feels like macOS

SwiftUI windows, menu bar extra, Finder reveal, launch-at-login — a dependency-free native Swift package.

📤

Bookmarks & export

Save searches as bookmarks and user filters, replay history, and export results to CSV, TXT, or file lists.

Speak fluent Everything.

The query language you already know works out of the box.

# files only, by extension
ext:jpg;png size:>1mb
# grouped OR with a type filter
<invoice | receipt> pdf dm:lastmonth
# regex, duplicates, ranges
regex:^report_\d+ dupe: len:3..12 width:800..1920
# formula-style property comparisons
$date-modified:==$date-created: child-count:>100

Stop waiting for Spotlight.

Download the DMG, drop it in Applications, and search at the speed of thought.
